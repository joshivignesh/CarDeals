# 🚗 CarDeals — Automotive Marketplace Platform

[![.NET](https://img.shields.io/badge/.NET-512BD4?style=flat-square&logo=dotnet&logoColor=white)](https://dotnet.microsoft.com)
[![React](https://img.shields.io/badge/React-61DAFB?style=flat-square&logo=react&logoColor=black)](https://react.dev)
[![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat-square&logo=typescript&logoColor=white)](https://typescriptlang.org)
[![MongoDB](https://img.shields.io/badge/MongoDB-47A248?style=flat-square&logo=mongodb&logoColor=white)](https://mongodb.com)

A modern, full-stack automotive marketplace application for browsing, listing, and managing car deals. Built with a clean RESTful API backend and a responsive React frontend.

## ✨ Features

- 🔍 **Search & filter** cars by make, model, price range, and year
- 📋 **Detailed listings** with specs, images, and seller contact info
- 🔐 **Authentication** — JWT-based login and registration
- 💼 **Dealer dashboard** — manage listings, track enquiries
- 📱 **Responsive design** — mobile-first UI

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React · TypeScript · Tailwind CSS |
| Backend | ASP.NET Core / Node.js · REST API |
| Database | MongoDB |
| Auth | JWT Bearer Tokens |
| Deployment | Docker |

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/joshivignesh/CarDeals.git
cd CarDeals

# Install dependencies
npm install

# Start development server
npm run dev
```
